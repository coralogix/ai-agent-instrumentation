# Contributing Guide

Welcome! We are glad that you want to contribute to this project.

## Ways to Contribute

- Bug fixes and improvements
- New AI agent integrations
- Dashboard enhancements
- Documentation updates

## Pull Request Lifecycle

- Open a PR; if it's not finished yet, please make it a draft first
- Reviewers will be assigned and aim to respond within 2 days

## CLA

We require that contributors have signed our Contributor License Agreement (CLA).

When a contributor submits their first Pull Request, the CLA Bot will request them to sign the [Coralogix CLA](https://cla-assistant.io/coralogix/ai-agent-instrumentation).

## Pull Request Checklist

- CLA signed
- Code follows existing patterns in the repo
- README updated if behaviour or configuration changes
- No secrets or credentials committed (`.env` is git-ignored)

## Ask for Help

Open a GitHub issue and we will get back to you as soon as possible.
